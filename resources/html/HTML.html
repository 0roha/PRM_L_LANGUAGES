WHAT IS HTML?

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser.

It provides the structure of a webpage and is composed of elements that tell the browser how to display content.

BASIC STRUCTURE

<!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>
  <h1>My First Heading</h1>
  <p>My first paragraph.</p>
</body>
</html>

TEXT ELEMENTS

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<p>Paragraph</p>
<a href="https://example.com">Link</a>
<strong>Bold text</strong>
<em>Italic text</em>

LISTS

<!-- Unordered List -->
<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>
<!-- Ordered List -->
<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>

IMAGES & MEDIA

<img src="image.jpg" alt="Description">
<!-- Audio -->
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>
<!-- Video -->
<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
</video>

FORMS

<form action="/submit" method="post">
  <label for="fname">First name:</label>
  <input type="text" id="fname" name="fname">
  <input type="submit" value="Submit">
</form>

SEMANTIC HTML

<header>Intro content</header>
<nav>Navigation links</nav>
<main>Main content</main>
<section>Thematic grouping</section>
<article>Self-contained content</article>
<footer>Closing content</footer>
