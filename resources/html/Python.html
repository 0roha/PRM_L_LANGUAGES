1. Introduction

Python is a high-level, interpreted language valued for readability and rapid development. Uses: web, data, automation, AI, scripting.

2. Basic Syntax

Indentation defines blocks. No semicolons required.

print("Hello, World!")  # simple output

3. Variables & Data Types

Python is dynamically typed. Common built-ins shown below.

# Numbers
x = 10          # int
y = 3.14        # float

# Strings
name = "Alice"

# Boolean
is_valid = True

# Containers
lst = [1,2,3]
tpl = (1,2)
d = {"k": "v"}
st = {1,2,3}

4. Operators

# Arithmetic
a = 5 + 3
b = 5 ** 2   # exponent
c = 10 // 3  # floor div

# Comparison
5 > 3   # True

# Logical
True and False  # False

# Membership
"a" in "apple"  # True

5. Input & Output

name = input("Enter your name: ")
print("Hello,", name)

6. Control Flow

If / Else

x = 10
if x > 0:
    print("Positive")
elif x == 0:
    print("Zero")
else:
    print("Negative")

Loops

# for loop
for i in range(5):
    print(i)

# while loop
n = 5
while n > 0:
    print(n)
    n -= 1

7. Functions

def greet(name):
    return "Hello, " + name

print(greet("Alice"))

8. Data Structures

# List
fruits = ["apple","banana"]
fruits.append("orange")

# Tuple (immutable)
point = (3,4)

# Set
s = {1,2,3}

# Dict
person = {"name":"Alice","age":25}

9. Strings

text = "Python"
print(text.upper())    # PYTHON
print(text[0:3])       # Pyt
print(len(text))       # 6

10. Exception Handling

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")
finally:
    print("Done")

11. File Handling

# Write
with open("data.txt","w") as f:
    f.write("Hello Python")

# Read
with open("data.txt","r") as f:
    print(f.read())

12. Object-Oriented Programming

class Animal:
    def __init__(self, name):
        self.name = name
    def speak(self):
        print(self.name + " makes a sound")

class Dog(Animal):
    def speak(self):
        print(self.name + " barks")

dog = Dog("Buddy")
dog.speak()

13. Modules & Libraries

Use import for built-ins and pip install for third-party packages.

import math
print(math.sqrt(16))  # 4.0

14. Advanced Concepts

Comprehensions, Lambdas, Generators

# List comprehension
squares = [x**2 for x in range(5)]

# Lambda
f = lambda x: x + 2

# Generator
def my_gen():
    yield 1
    yield 2
