The fundamentals of C++ programming encompass several core concepts essential for building functional and efficient applications.

Basic Syntax & Structure

The foundation of every C++ program:

  • Header files and namespaces
  • Main fution structure
  • Comments and documentation
  • Preprocessor directives
// Basic C++ program structure
#include <iostream>
using namespace std;

int main() {
  cout << "Hello, Cyber World!" << endl;
  return 0;
}

Variables & Data Types

Storing and manipulating data:

  • Primitive data types
  • Type modifiers and conversion
  • Constants and literals
  • Type inference with auto
int score = 100;
double precision = 3.14159;
char grade = 'A';
bool isCyber = true;
auto future = 2077; // type inferred as int

Operators

Performing operations on data:

  • Arithmetic operators
  • Relational and logical operators
  • Bitwise operators
  • Assignment and compound operators
int a = 10, b = 5;
int sum = a + b; // 15
bool result = (a > b) && (a != 0); // true
a += 10; // a becomes 20
int bits = a & b; // bitwise AND

Input & Output

Interacting with the user:

  • Standard output with cout
  • Standard input with cin
  • Formatting output
  • File I/O operations
int age;
string name;
cout << "Enter your name: ";
cin >> name;
cout << "Hello " << name << "! Welcome to 2077" << endl;

Control Flow

Directing program execution:

  • Conditional statements (if/else)
  • Switch statements
  • Loop structures (for, while, do-while)
  • Break and continue statements
for (int i = 0; i < 10; i++) {
  if (i % 2 == 0) {
    cout << i << " is even" << endl;
  } else {
    continue;
  }
}

Functions

Modular and reusable code:

  • Function declaration and definition
  • Parameters and return types
  • Function overloading
  • Recursion and inline functions
int add(int a, int b) {
  return a + b;
}

double add(double a, double b) {
  return a + b; // function overloading
}

Arrays & Pointers

Working with memory and collections:

  • Array declaration and initialization
  • Pointer variables and memory addresses
  • Dynamic memory allocation
  • Pointer arithmetic
int arr[5] = {1, 2, 3, 4, 5};
int* ptr = arr; // pointer to first element
cout << *ptr; // outputs 1
ptr++; // move to next element
cout << *ptr; // outputs 2

OOP Concepts

Object-oriented programming principles:

  • Classes and objects
  • Encapsulation and abstraction
  • Inheritance and polymorphism
  • Constructors and destructors
class Android {
  private:
    string model;
  public:
    Android(string m) : model(m) {}
    void speak() {
      cout << "I am model " << model << endl;
    }
};
