INTRODUCTION TO PHP

PHP (Hypertext Preprocessor) is a server-side scripting language designed for web development. It's powerful, flexible, and used by millions of websites.

Hello, Cyber World!
// PHP code is executed on the server

PHP was created by Rasmus Lerdorf in 1994 and has evolved into a full-featured language that powers major platforms like WordPress, Wikipedia, and Facebook.

Why PHP?

  • Runs on almost all servers and operating systems
  • Compatible with almost all databases
  • Open source with a huge community
  • Easy to learn with great documentation

PHP SYNTAX

PHP code is embedded within HTML and executed on the server before the page is sent to the client's browser.

// PHP code is enclosed in  tags

Key Syntax Rules:

  • All PHP code must be enclosed within tags
  • Statements must end with a semicolon (;)
  • Variable names start with a dollar sign ($)
  • PHP is case-sensitive for variable names
  • Comments can be // for single-line or /* */ for multi-line
